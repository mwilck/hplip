Technical Documentation: Man Pages

hplip-install : Installer for HPLIP tarball.

hp-setup : Installs HPLIP printers and faxes in the CUPS spooler. Tries to automatically determine the correct PPD file to use. Allows the printing of a testpage. Performs basic fax parameter setup.

hp-probe : Discover USB, parallel, and network printers.

hp-makeuri : Creates device URIs for local and network connected printers for use with CUPS.

hp-toolbox : The HP Device Manager (aka Toolbox) for HPLIP supported devices. Provides status, tools, and supplies levels.

hp-align : Cartridge alignment utility for HPLIP supported inkjet printers.

hp-check : Check the existence and versions of HPLIP dependencies.

hp-clean : Cartridge cleaning utility for HPLIP supported inkjet printers.

hp-colorcal : Perform color calibration on HPLIP supported inkjet printers. (Note: Not all printers require the use of this utility).

hp-fab : A simple fax address book for HPLIP.

hp-info : Query a printer for both static model information and dynamic status.

hp-levels : Display bar graphs of current supply levels for supported HPLIP printers.

hp-print : A simple front end to 'lp'. Provides a print UI from the Device Manager if kprinter, gtklp, or xpp are not installed.

hp-sendfax : Allows for sending faxes from the PC using HPLIP supported multifunction printers.

hp-testpage : Print a tespage to a printer. Prints a summary of device information and shows the printer's margins.

hp-unload : Access inserted photo cards on supported HPLIP printers. This provides an alternative for older devices that do not support USB mass storage or for access to photo cards over a network.

hp-makecopies : PC initiated make copies on supported HP AiO and MFP devices.

hp-timedate : Set the time and date on an HP Officejet.

hp-firmware : Download firmware to a device.

hpssd : Provides persistent data and event services to HPLIP client applications.

hp-scan : SANE-based scan utility for HPLIP.
