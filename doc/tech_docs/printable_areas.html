    Technical Documentation: Printable Areas

    Printable area or margins are device class and print mode dependent. For example inkjet products, when Duplex mode is set the top margin is set to "1/2" inch. Normally the top margin is "1/8" inch. Laserjet products do not have different margins during duplex.

    Printers that support full bleed will require a slightly oversize printable width and height to account for over spray and paper skew.

    The printable area values are available over the IJS interface. Ghostscript uses this data during postscript to raster conversion.

    The following tables (in inches and mm) list the physical printable area for all paper sizes supported by HPIJS.
