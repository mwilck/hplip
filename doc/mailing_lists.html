Mailing Lists

Two mailing lists are available to get help, report bugs, make suggestions, and to discuss the HPLIP project.

If you are experiencing a hardware problem with your HP printer, you will need to contact HP's regular support services (Note: this link is for US/English support at hp.com. You may need to change this setting at hp.com for your language and region/country). Unfortunately, we are unable to assist with hardware problems on these mailing lists.

Note

For Windows and Macintosh users seeking drivers and/or support please go to http://www.hp.com.

Important

When posting to the mailing list please send this information along with your report:

1. Your linux distro and version.
2. Your printer make/model.
3. If possible run 'hp-check -t' and attach the generated hp-check.log file to your email.

This is a moderated mailing list. Your email may not post to the list until it is approved by a moderator.

The HPLIP-Help Mailing List

This mailing list is the primary place for end users to get help with HPLIP issues and problems.

  • Post a message to the HPLIP-Help list
  • Subscribe to the HPLIP-Help list
  • View the HPLIP-Help list archive

Note: This list is also available as a newsgroup by using the Gmane service at: http://dir.gmane.org/gmane.comp.printing.hplip.user

The HPLIP-Devel Mailing List

The HPLIP-Devel list is reserved for more technical discussions of bugs, enhancement requests, issues reported by the distributions, etc. Please do not post end user issues on this list.

  • Post a message to the HPLIP-Devel list
  • Subscribe to the HPLIP-Devel list
  • View the HPLIP-Devel list archive

Note: This list is also available as a newsgroup by using the Gmane service at: http://dir.gmane.org/gmane.comp.printing.hplip.devel
