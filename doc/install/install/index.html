Installer Walkthrough

Step 1: Download the automatic installer.

Download HPLIP 2.8.2

Download the Self-Extracting Archive With Installer

Note

After downloading make sure to put the hplip-2.8.2.run file in the directory you will want the hplip directory to be created. You may want this to be in your /home/username/ directory rather than your desktop.

Step 2: Run the automatic installer.

Open a terminal or command line window. Then run:

sh hplip-2.8.2.run

At this step the hplip directory is created and the hplip source files are extracted. You should see something similar to the below.

Step 3: Select the install process you'd like to run.

Select the install mode: Automatic or Custom.

Note

In automatic mode the following settings are preset: Installs HPLIP, doesn't display the release notes, installs all options EXCEPT parallel support, doesn't ask to confirm linux distribution, tries to remove previous installs of HPLIP and HPOJ, assumes that all optional and required dependencies will be downloaded and installed, installs HPLIP to the /usr directory, will automatically compile and install hplip, and will run the hp-setup tool at the end to configure your printer.

Warning

In automatic mode parallel support is NOT enabled.

Step 4: Verify your linux distribution.

Note

Ensure the installer detected the correct linux distribution. Different linux versions use different install procedures if the incorrect version is selected the installer may fail.

Step 5: Enter your root user password.

Step 6: Read the installation notes.

Depending on your linux distro some steps may need to be completed before going further with the HPLIP install. Please follow any directions given.

Step 7: Missing dependencies are detected and package manager clean up is preformed.

The installer will now detect what dependencies your system will need to install before installing HPLIP. In addition any package manager clean up steps will be ran. These are executed to try and minimze any potential package manager problems.

Step 8: The missing dependencies are download and installed.

If you system is missing any dependencies they are now downloaded and installed.

Step 9: ./configure and make are executed.

./configure prepares HPLIP for install as well as your system is verified to have all the required dependencies for HPLIP.

make is then executed. Make compiles HPLIP for your system.

Step 10: make install is executed.

make install is the actual install procedure, this step is where HPLIP is actually being installed to your system.

Step 11: Final install steps are ran.

After the install is completed the HPLIP and CUPS daemons are restarted so that HPLIP will function correctly. A daemon is simply an application running in the background, also known as a service.

Finally the hp-setup utility is executed. The hp-setup application will assist you in installing and configuring your printer. You may be prompted to enter your root passoword at this step.
