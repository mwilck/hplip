    Installation - Step 4 - CUPS - USB/Parallel

    1. Connect your HP printer to your computer using USB or Parallel.

    Note: See your printer documentation for specific instructions on connecting the USB, Parallel or Network cable to your printer and computer.

    1. Open a console/terminal window.
    2. Enter this command to login as super user; (Ubuntu users skip this step)
    $ su

    Warning: The root password gives you administrative privileges on the system.

    1. Enter this command to restart the Common UNIX Printing system (CUPS)
    # /etc/init.d/cups restart

    On some systems you may need to enter this command: (Fedora Core for example)

    # /etc/init.d/cupsys restart
    1. Ubuntu Users: Enter this command:
    # sudo /etc/init.d/cups restart
    1. Open a web browser, and then enter the following address in the address box: http://localhost:631
    2. Click Printers -> Add Printer
    1. Enter the root username and password. (when prompted)
    2. Enter a name for your HP printer in the Name box.

    Note: The name you enter must not contain spaces.

    1. (Optional) Enter a description of the location of the printer in the Location box. (Example: 3rd Floor Office)
    2. (Optional) Enter a description of the printer in the Description box. (Example: HPLJ3380)
    3. Click Continue.
    4. From the Device drop-down list select the printer to be installed.

    Note: You MUST make a selection from the drop down that appears such as "hp:/usb/hp_deskjet_5600?serial=MY31R1K02179".

    hp:/usb/[printer name, serial number]
    1. Click Continue.
    2. In the Make list, click HP -> Continue.
    1. In the Model list, click the printer model that most closely matches your printer, and then click Continue.

    Note: To configure your printer for using Fax, follow these steps, otherwise goto Step 27.

    1. Once the printer is added using the above steps, go back to step 7.
    2. At step 9, enter a name for your Fax device. Example: "photosmart_3310_fax"
    3. At step 13, select your device that starts with "hpfax://".
    Example: "hpfax:/usb/psc_2500_series?serial=XXXXXXXXX (HP psc_2500_series)
    1. At step 15, select HP
    2. At step 16, scroll to the very bottom and select "HPLIP Fax (en)"
    3. Complete the setup process.
    4. Open the hp-toolbox. (Open a terminal/console window and enter)

    Note: You can also open the HP Device Manager by using the GNOME or KDE menu system.

    # hp-toolbox

    Note: If setting up fax continue else go to step 30.

    1. Select your printer.
    2. Click the "Tools & Settings" tab.
    3. Click "Setup Fax.."
    4. Enter your information in the "Fax Header Information" section.
    5. Click "Ok".
    6. Fax setup is now complete.
    1. Click the Tools & Settings tab, and then click Print Test Page.

    Having trouble? Check the Installation/Troubleshooting Help section for assistance!
