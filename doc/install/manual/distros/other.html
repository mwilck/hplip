Other Linux Versions

Note: Steps and dependencies will depend on your linux distribution, please refer to your linux distribution documentation and/or support resources for possible assistance.

Note: Your package names will vary depending on your linux distribution.

First, ensure that your system meets the HPLIP minimum system requirements before proceeding.

Link to the HPLIP Minimum System Requirements

Second, install the HPLIP Required Dependencies:

Dependencies Description
libcrypto OpenSSL cryptographic library
gcc GNU Project C and C++ Compiler
SANE Scanning library
GhostScript PostScript and PDF language interpreter and previewer.
libjpeg JPEG library
libpthread POSIX threads library
make GNU make utility to maintain groups of programs
python-devel Python development files
Reportlab PDF library for Python
PyQt Qt interface for Python
cups-devel Common Unix Printing System development files
ppdev Parallel port support kernel module
libusb USB library
scanimage Shell scanning program
libnetsnmp-devel SNMP networking library development files
Python 2.2 or greater Python programming language
LSB Linux Standard Base support
xsane Graphical scanner frontend for SANE
cups Common Unix Printing System
Python 2.3 or greater Required for fax functionality
