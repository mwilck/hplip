Other Linux Distros and/or Manual Installation

Step 1

Ensure that your system meets the HPLIP minimum system requirements before proceeding.

Step 2

Install these HPLIP dependencies using your distribution's package management tool:

(Note: Your package names will vary depending on your Linux distribution. please refer to your Linux distribution documentation and/or support resources.)

Dependency Description Required?
libcrypto OpenSSL cryptographic library Yes
gcc GNU Project C and C++ Compiler Yes
GhostScript PostScript and PDF language interpreter and previewer. Yes
libjpeg JPEG library Yes
libpthread POSIX threads library Yes
make GNU make utility to maintain groups of programs Yes
cups-devel Common Unix Printing System development files Yes
libusb USB library Yes
python2.2+ Python programming language version 2.2 or greater Yes
LSB Linux Standard Base support Yes
cups Common Unix Printing System Yes
python-devel Python development files Yes
Reportlab PDF library for Python No, optional for fax cover pages
PyQt Qt interface for Python No, optional for GUI
ppdev Parallel port support kernel module No, optional for parallel port I/O
scanimage Shell scanning program No, optional for scanning support
libnetsnmp-devel SNMP networking library development files No, optional for networking support
xsane Graphical scanner frontend for SANE No, optional for scanning support
python2.3+ Python programming language version 2.3 or greater No, optional for fax support
SANE SANE scanning library No, optional for scanning support
python-imaging Scanning library No, optional for scanning support

Step 3

Download HPLIP

(Note: View the current Release Notes for additional information on this release.)

Step 4

For specific information on using ./configure and the different options please go here. Also reference your distro documentation and configuration for additional information on which tags are relevent.

Step 5

Run 'make' in a terminal shell in the HPLIP source directory to compile HPLIP.

Step 6

Run 'make install' in a terminal shell as root in the HPLIP source directory to install HPLIP.

Step 7

If you are going to be setting up a printer with USB and the printer is already plugged in, you will need to do one of the following:

  1. Unlpug and re-plug in the USB printer, or
  2. Restart your PC.

Failure to do this will cause your USB printer to not be detected by HPLIP.

Step 7

Setup your printer (using hp-setup (recommended), CUPS or Yast)
