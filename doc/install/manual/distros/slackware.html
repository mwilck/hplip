Slackware

Note: Slackware has not been tested, however assuming that your system meets the minimum system requirements HPLIP should function correctly.

Most successfull installs have been with using HPIJS only and not the full HPLIP.

See this document for information on how to install HPIJS. HPIJS provides only the printing functionality for your printer, all other extended functionality will not function with HPIJS. ie, scanning, photo card, etc.

Review the HPLIP system requirements for more information.

View the current Release Notes for additional information on this release.

Note: After you have complete the install procedure you'll want to excute the following steps. Please do this after you have finished the install process.

cd /etc/init.d

mv hplip /etc/rc.d/rc.hplip

cd /

rm -r /etc/init.d
