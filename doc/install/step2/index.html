Installation - Step 2 - Setup the Linux environment

You must install the prerequisite packages in order for the HP printing software to install and operate.

Note: Have the installation CDs for your Linux distribution available as you set up the Linux environment. You might be prompted for the installation CDs as the system searches for the necessary packages.

Follow the steps for the Linux distribution you are using:
