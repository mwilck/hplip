Installation - Step 2 - Mandrake/Mandriva

  1. Open a console/terminal window.
  2. Login as the super user (root), enter this command:

Warning: The root password gives you administrative privileges on the system.

su
  1. Press Enter.
  2. Enter the root password.
  3. Press Enter.
  4. Follow these steps to download the appropriate urpmi packages:

In your browser goto the Easy URPMI repository configuration tool for Mandriva web site: http://easyurpmi.zarb.org

Configure your system using the steps provided by the Easy URPMI website. Make sure to configure the Source contrib and the Source main repositories.

  1. Follow this step to install the necessary packages:
  2. You will now need to deactivate the Mandrica CD's from the Software Media Manager
Click "K-Menu" > System > Configuration > Packaging > Software Media Manager
  1. Remove the check marks next to the Mandriva CD's in the "Enabled?" column.
  2. Now enter this command:
urpmi gcc-c++ cups cups-devel python-devel PyQt net-snmp-mib net-snmp-devel libjpeg-devel xsane

The appropriate packages should be downloaded and installed.

Proceed to step 3.
