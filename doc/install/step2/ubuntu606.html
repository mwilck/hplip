    Installation - Step 2 - Ubuntu 6.06

    1. Open a console/terminal window.
    2. Enter this command:
    sudo apt-get install build-essential python2.4-dev python2.4-qt3 libcupsys2-dev libsnmp9-dev libjpeg62-dev lsb libtool automake1.9 libusb-dev
    1. If you are prompted for the PostFix configuration, select No Configuration, or select the option that best works with your system configuration.

    Proceed to step 3.
