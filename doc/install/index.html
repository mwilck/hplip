    Installation

    Important: Before proceeding, check Supported Devices to determine if HPLIP will work with your device.

    There are two different ways to install HPLIP.

    Installer: This is the new installer for HPLIP. The It's a more automated process which will greatly speed up your install. These distros are currently supported:

    • SUSE Linux (10.0, 10.1)
    • Fedora (3.0, 4.0, 5.0, 5.92)
    • Ubuntu (5.04, 5.1, 6.06, 6.10)
    • Debian (2.2, 3.0, 3.1)
    • Mandriva Linux (2006.0, 2007.0)

    or you can use the four step process outlined below.

    Step 1: Verify if HPLIP is already installed in your Linux distribution.

    Step 2: Set up the Linux environment.

    Step 3: Download and install the HP printing software.

    Step 4: Setup your printer.

    Note: In order to install the HP printing software, you must have super user (root) privileges on the computer. If you do not have super user privileges, contact your system administrator.
