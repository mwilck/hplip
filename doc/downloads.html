Download HPLIP 2.7.10

HPLIP is available in two download types. Please choose the download that best matches your requirements.

HPLIP Self Extracting Archive With Installer

This is an experimental release that packages a self-extracting installer together with the regular tarball. The self installer is know to work on the following distributions (see note 2):

  • SUSE Linux (10.0, 10.1, 10.2, 10.3)
  • Fedora (3.0, 4.0, 5.0, 5, 5.92, 6.0, 6, 7.0)
  • PCLinuxOS (2006.0, 2007.0)
  • IGOS (1.0)
  • Ubuntu (5.04, 5.1, 6.06, 6.10, 7.04, 7.10)
  • Debian (2.2, 3.0, 3.1, 4.0, 4.0r0)
  • Mepis (6.0, 6.5)
  • Mandriva Linux (2006.0, 2007.0, 2007.1)

To use the self-extracting file, follow these steps:

  1. Download the file to a convenient location (e.g., home directory or desktop, etc).
  2. Open a console/terminal and cd to the download directory.
  3. Type in and run this command: 'sh hplip-2.7.10.run'

Download the Self-Extracting Archive With Installer

Notes:

  1. You will need to have root/su access to run the installer.
  2. The installer will work on other distributions, but the automatic installation of dependencies will not be available.

CAVEATS AND KNOWN ISSUES

  1. At times during processing, the installer may seem to pause for significant amount of time. If this occurs, please enter your root (su/sudo) password and hit enter. This issue will be addressed in upcoming releases.
  2. The installer will report in its banner the version of the currently installed HPLIP (if installed), rather than the newly being installed version.
  3. If the installer fails for function for you, please submit a bug report to the mailing list, and use the regualar HPLIP tarball and manual install method.

HPLIP Regular Tarball

This is the traditional method to download and install HPLIP. It requires that you manually install dependencies and run build and install commands in a console/terminal. To use it, follow the standard directions at Installation Instructions

Download the Regular Tarball

Notes:

  1. The regular tarball also comes with the installer, but it must be run manually if desired (instead of using the traditional installation approach). To run the installer manually, after extracting the tarball, cd into the hplip-2.7.10 directory and run hplip-install.
